# INE5660 - Projetos II (SIN/INE/CTC/UFSC)

Este repositório tem como objetivo disponibilizar os códigos utilizados ao longo do Trabalho de Conclusão de Curso intitulado "Avaliação do potencial da linguagem de programação Rust no desenvolvimento de aplicações científicas de alto desempenho"

## Requisitos

Para a correta execução dos experimentos, torna-se necessário as seguintes ferramentas:

1. Rust
2. Cargo
3. gfortran
4. perf

Para conferência utilize os comandos:

```
rustc --version
cargo --version
gfortran --version
perf --version
```
